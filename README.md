# Amazon-Business-Performance-PowerBI-Dashboard
Amazon Business Performance Dashboard (Power BI)

Project Overview:
This Power BI dashboard analyzes Amazon retail business performance using
transactional data modeled in a star schema format.

The dashboard provides insights into revenue, profit, profit margin,
product performance, regional sales, and country-wise revenue trends.

Data Model:
- Fact_Transactions: Sales quantity, revenue, transaction dates
- Fact_Returns: Product returns
- Dim_Customers: Customer demographics
- Dim_Products: Product details and pricing
- Dim_Stores: Store and location information
- Dim_Regions: Regional hierarchy
- Dim_Calendar: Date-based analysis (Year, Month, Quarter)

Tools & Skills Used:
- Power BI Desktop
- Power Query (data cleaning & transformation)
- Data Modeling (Star Schema)
- DAX measures (Revenue, Profit, Profit Margin, YTD Sales)
- Interactive slicers (Month, Year)

Key Insights:
- Identified high-performing states and countries by revenue
- Analyzed profit vs previous month performance
- Evaluated product brand contribution to revenue and profit
- Tracked YTD sales and profit margins

Business Value:
Helps business stakeholders monitor performance, identify growth regions,
optimize product strategy, and make data-driven decisions.
